# Sales Data Analysis Project

## Objective
Analyze sales data to identify trends, top customers, and key business insights
using Excel, MySQL, Power BI, and Python.

## Tools Used
- Excel – Data cleaning, pivot tables, and basic dashboard
- MySQL – Data storage and SQL analysis
- Power BI – DAX measures and interactive dashboard
- Python – Data analysis and visualizations

## Project Workflow
- Cleaned and prepared sales data using Excel
- Created pivot tables and Excel dashboard
- Performed data analysis using SQL in MySQL
- Built interactive dashboards using Power BI
- Analyzed data and created graphs using Python
- Documented the complete project in a PDF report

## Project Contents
- Excel files: cleaned data, pivot tables, dashboard
- MySQL scripts: table creation and queries
- Power BI file (.pbix): sales dashboard
- Python file: data analysis and graphs
- PDF file: complete project documentation
